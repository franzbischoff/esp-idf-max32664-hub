8 This is an Arduino Library written for the MAXIM 32664 Biometric Sensor Hub
9 The MAX32664 Biometric Sensor Hub is in actuality a small Cortex M4 microcontroller
10 with pre-loaded firmware and algorithms used to interact with the a number of MAXIM
11 sensors; specifically the MAX30101 Pulse Oximter and Heart Rate Monitor and
12 the KX122 Accelerometer. With that in mind, this library is built to
13 communicate with a middle-person and so has a unique method of communication
14 (family, index, and write bytes) that is more simplistic than writing and reading to
15 registers, but includes a larger set of definable values.
